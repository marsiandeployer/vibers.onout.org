Privacy Policy

Last updated: April 7, 2026 · Applies to the Vibers Review GitHub App

What We Are

Vibers Review ("we", "us") is a human-in-the-loop code review service operated by Onout (onout.org). We provide code review and pull request fixes for GitHub repositories through our GitHub App.

What Data We Collect

When you install the Vibers Review GitHub App, we receive and process:

  • Repository content — source code from repositories you grant us access to, for the sole purpose of performing code review
  • GitHub account information — your GitHub username, account ID, and installation ID (provided automatically by GitHub)
  • Commit metadata — author, timestamp, commit message, changed files list
  • Setup form data (optional) — spec URL, Figma URL, contact info you voluntarily provide after installation
  • Webhook events — push events and installation events sent by GitHub to our server

We do not collect passwords, payment information, or private environment variables. We cannot see files that are not committed to the repository.

How We Use Your Data

  • To perform code review against your specification
  • To open pull requests with fixes in your repository
  • To contact you regarding the review (via the contact info you provide)
  • To improve our review process internally

We do not sell, share, or disclose your data to third parties, train AI models on your code, or use your repository content for any purpose other than performing the requested review.

Data Retention

We retain repository content only as long as necessary to complete a review. Installation records (GitHub account, installation ID) are retained as long as the App is installed. You can request deletion at any time by contacting us.

Third-Party Services

Our server uses standard infrastructure. We do not share code content with third-party AI services, analytics platforms, or data brokers.

Your Rights

You can revoke the App's access at any time: GitHub Settings → Applications → Installed GitHub Apps → Vibers Review → Uninstall. This immediately removes all repository access.

To request deletion of any stored data, email us at support@onout.org.

Security

Access to your code is restricted to reviewers working on your specific review. We use webhook signature verification to ensure all GitHub events are authentic. Our server does not store repository clones after the review is complete.

Changes to This Policy

We may update this policy. The "Last updated" date at the top reflects the most recent revision. Continued use of the App after changes constitutes acceptance.

Contact

Questions? Email support@onout.org or open an issue at github.com/apps/vibers-review.
